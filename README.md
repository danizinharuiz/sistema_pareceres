 Sistema Web de Pareceres Institucionais

 Aplicação web desenvolvida para automatização e padronização
de pareceres administrativos institucionais.

O sistema permite preenchimento dinâmico de informações,
cálculo automático de carga horária semanal e geração de PDF
para emissão de pareceres.

Funcionalidades

- Preenchimento dinâmico de pareceres
- Campos editáveis integrados ao texto
- Cálculo automático de carga horária semanal
- Inserção automática da data de emissão
- Geração de PDF via impressão do navegador
- Layout institucional responsivo
- Navegação entre modelos de pareceres

  Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap

  Como utilizar

1. Abra o arquivo HTML no navegador
2. Preencha os campos necessários
3. Clique em "Gerar PDF"
4. Escolha a opção "Salvar como PDF"

   Objetivo

O projeto foi desenvolvido com o objetivo de otimizar
a emissão de pareceres administrativos, reduzir retrabalho,
padronizar documentos e melhorar o fluxo de preenchimento
institucional.
